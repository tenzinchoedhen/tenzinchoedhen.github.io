The Simon-Ehrlich Bet: A Data Analysis (1980-1990)

Time Series Analysis & Economic Data Visualization using R and Federal Reserve Data

  • Complete R analysis code

Project Overview

Objective

Analyze historical commodity price data to evaluate the famous wager between economist Julian Simon and biologist Paul Ehrlich regarding resource scarcity.

Tools Used

R (tidyverse, ggplot2, pdfetch, dygraphs)

Key Skills Demonstrated

  • Time series data processing and visualization
  • Inflation adjustment using Consumer Price Index data
  • Multi-source data integration
  • Statistical analysis and economic interpretation

The Analysis

Background

In 1980, Julian Simon wagered that the inflation-adjusted prices of five strategic metals (copper, chromium, nickel, tin, and tungsten) would decrease over the next decade, contradicting Paul Ehrlich's prediction of increasing resource scarcity.

Data Sources

  • Metal Prices: U.S. Geological Survey historical data (1840-2010)
  • Inflation Adjustment: Federal Reserve CPI data (FRED database)
  • Analysis Period: 1980-1990

1. Data Import and Processing

# Import metal price data from USGS
tungsten <- import("https://pubs.usgs.gov/sir/2012/5188/tables/tungsten.xlsx", skip = 2)
copper <- import("https://pubs.usgs.gov/sir/2012/5188/tables/copper.xlsx", skip = 2)
# ... (chromium, nickel, tin)

# Convert to time series objects
tungsten.ts <- ts(tungsten, start=c(1959), end=c(2010), frequency=1)
copper.ts <- ts(copper, start=c(1850), frequency=1)
# ... (chromium, nickel, tin)

2. Inflation Adjustment

# Download CPI data from FRED
cpi <- pdfetch_FRED("CPIAUCSL")
cpi.annual <- window(cpi.ts, start = c(1959), end = c(2010), frequency = 1)

# Calculate real (inflation-adjusted) prices
real_P_tungsten = tungsten.ts * 100 / cpi.annual
real_P_copper = copper.ts * 100 / cpi.annual
# ... (remaining metals)

Results

Key Visualizations

Figure 1: Real Prices During the Bet Period: All five metals showed declining inflation-adjusted prices from 1980-1990. The composite index (dashed line) fell by 32.2%.

Figure 2: Rebased Index (1980 = 100): When normalized to a baseline of 100 in 1980, all metals finished below the starting point by 1990.

Winner

Julian Simon collected $576.07 from Paul Ehrlich in October 1990, as all five metals declined in real price.


Insights

Market Forces Driving Price Declines:

Supply-Side Factors:

  • New mining technologies reduced extraction costs
  • Discovery of new deposits expanded global reserves
  • Improved processing efficiency increased yields
  • Post-Cold War supply expansion from former Soviet bloc

Demand-Side Factors:

  • Early 1980s recessions reduced industrial demand
  • Materials science advances enabled substitution (e.g., fiber optics replacing copper)
  • Manufacturing efficiency improvements reduced material intensity
  • Electronic miniaturization decreased per-unit requirements

Conclusion

This analysis demonstrates how market mechanisms and technological innovation responded to perceived resource scarcity during the 1980s. While Simon won the specific bet, the broader debate about resource sustainability continues to inform contemporary policy discussions around critical minerals, renewable energy and environmental limits.
